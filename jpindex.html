内容: このウェブサイトは、若者コミュニティ「集う」のサイトです。様々なイベントを提供しており、目的は楽しむことと新しい友達を作ることです。また、参加費は「嵐山子ども食堂」を通じて、温かくおいしい食事を提供するための寄付に使われます。"

言語: HTML CSS JavaScript

工夫した点: ウェブサイトのイベントページで、動画を背景画像の桜の花の代わりに、ページの読み込み時間の削減とより視覚的な質に仕上げるため、CSSアニメーションの「translate」プロパティを活用し、画像を「transform: translate(-70%, -70%)」から「transform: translate(100%, 100%)」に移動させることで、ダイナミックな動きを再現しました。こうすることで、動画よりも背景画像のファイルサイズが小さく、読み込み時間を短縮することができます。
